# ecommerce-backend

Backend base para un e-commerce modular y reutilizable.

## Stack

- Java 17 / 21
- Spring Boot 3.x
- Spring Data JPA
- PostgreSQL
- Flyway
- Hibernate Envers (auditoría)

## Requisitos

- JDK 17+
- Maven 3+
- Docker y Docker Compose
- Git

## Flujo de ramas

- `main`: rama estable
- `dev`: rama de desarrollo
- Ramas de feature: `feature/nombre-feature`

## Cómo clonar

```bash
git clone <url>
cd ecommerce-backend
git checkout dev
